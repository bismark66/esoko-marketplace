## Getting Started

### Prerequisites

Node.js (v16+)
npm or yarn

## Installation

```
# Clone the repository
git clone https://github.com/yourusername/vite-ecommerce.git

# Navigate to project directory
cd vite-ecommerce

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev

```
